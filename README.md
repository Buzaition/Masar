## 📌 Overview

**Masar (مَسار)** is a modern, comprehensive SaaS platform designed specifically for Egyptian lawyers and law firms. It streamlines daily legal workflows, from case management and session tracking to client communication and financial administration, all within a sleek, premium, and highly responsive user interface.

## ✨ Key Features

- **💼 Case Management**: Organize all your cases, documents, and client details in one centralized, secure location.
- **📅 Session Tracking**: Never miss a court date. Track upcoming sessions with intelligent reminders and timelines.
- **🏢 Law Firm Solutions**: Dedicated features for law firm administrators to manage multiple lawyers, delegate tasks, and monitor performance.
- **💰 Financial Tracking**: Keep track of fees, expenses, and billing effortlessly.
- **📱 Responsive & Premium Design**: A meticulously crafted user interface with glassmorphic elements, smooth micro-animations, and full RTL support, ensuring a world-class user experience on both desktop and mobile.

## 🚀 Tech Stack

- **Framework**: React 19 + Vite
- **Language**: TypeScript
- **Styling**: Tailwind CSS (with custom utility classes and premium aesthetic focus)
- **Routing**: React Router v7
- **State Management**: Zustand
- **Icons**: Lucide React
- **Charts**: Recharts

## 🛠️ Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone <repository-url>
   cd Lawyer
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```

## 🎨 Design Philosophy

Masar prioritizes **Visual Excellence**. The application moves away from generic, cluttered software interfaces and instead embraces curated, harmonious color palettes, modern typography, subtle gradients, and dynamic micro-animations to create an interface that feels responsive, alive, and extremely premium.
